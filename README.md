# Python Password Generator

This is a simple Python project that generates random passwords based on user preferences.

## Features
- Generate passwords of any length
- Option to include uppercase letters
- Option to include numbers
- Option to include symbols
- Random and secure password generation

## Technologies Used
- Python
- random module
- string module

## How It Works
1. The user enters the desired password length.
2. The user selects whether to include:
   - Uppercase letters
   - Numbers
   - Symbols
3. The program generates a random password based on the selected options.

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Run the Python file.
